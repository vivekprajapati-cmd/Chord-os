## Your tasks

Go to **Tasks** in the sidebar. Each card shows:
- Deliverable name and brand
- Priority and deadline
- Full brief — storyboard link, reference links, mood direction, what to avoid
- A **Briefing** badge if the task came from a client meeting

Every task should have a reference link. If it's missing, flag Nimesh before you start.

### Marking status
- **In Progress** — when you begin the edit
- **Ready for Review** — when first cut is done

Nimesh reviews after "Ready for Review."

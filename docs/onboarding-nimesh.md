## Day-to-day: how it works

### 1. Allocating video work via Chat

Go to **Allocator** in the sidebar. Type naturally:

> "Block Vineet 4 hours tomorrow for IndiaGate Eid reel edit, P1. Ref: [storyboard link]"

The AI confirms the block, checks for conflicts, and notifies Vineet on Slack.

You need to provide:
- Who (first name is enough)
- What (deliverable name)
- How long (hours)
- When (or it defaults to tomorrow 10am)
- At least one reference link (storyboard, mood board, drive link)

The system will ask for what's missing before creating anything.

### 2. Seeing your team's load

On the **Calendar** page — shows your blocks for the week. You can see day-by-day what's been assigned to you. For your team's load, check the **Tasks** page which shows all active tasks by status.

### 3. Briefings — what the client said

Every time a POC or lead logs a client meeting, it shows up in **Briefings**. Before starting any video work, check the briefing for that brand. You'll see:

- What the client decided
- High-impact action items
- The exact brief for each task

If a task was created from a meeting, it has a **Briefing** badge in the task list. The full brief is in the task card.
